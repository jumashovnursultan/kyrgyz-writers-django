# 📚 Кыргыз Жазуучулары

Веб-приложение о кыргызских писателях. Дипломная работа.

## Возможности
- Список писателей с поиском и фильтром по эпохе
- Детальная страница: биография, произведения, цитаты
- Случайный писатель 🎲
- Статистика по эпохам 📊

## Технологии
- Python / Django
- SQLite
- HTML + CSS

## Запуск
```bash
pip install django pillow
python manage.py migrate
python manage.py runserver
```
